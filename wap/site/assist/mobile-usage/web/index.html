About

There are many very good HTTP-log analyzers like AWstats, The Webalizer or Analog, just to name a few.
mobile-usage (the tool which generated this statistic) is not a full-blown log analyzer, just a supplement to existing ones with a special focus on mobile devices. It should give you answers to questions like:

  • How many visitors come to my site using phones or tablets?
  • Which mobile devices are used?
  • Which mobile operating systems and versions are used?
This statistic is based on a piece of information provided by browsers requesting web resources like HTML-pages and images. Web-Browsers and search engines usually send an HTTP-Header called User-Agent. With the help of Mobile-Detect, devices are detected by the User-Agent header.

You can view mobile-usage statistics by drag & drop the generated summary JSON file to this page!
